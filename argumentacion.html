Requerimientos

  1. Grilla responsiva: Utilizar la grilla de Bootstrap para distribuir los cupones en 1 columna para Mobile y 3 para Desktop.

    Se implemento de acuerdo a los solicitado utilizando una grilla de 3 cartas para PC y una carta para dispositivos pequeños, con el metodo y estructura BOOTSTRAP

  2. Clases de colores: Utilizar las clases de colores de Bootstrap que se asemejan a los colores de la maqueta.

    Con respecto a los colores se utilizaron colores similares para la barra de nevegacion, utilizando los colores con instrucciones de BOOTSTRAP y por otra parte para uno de los precios y el contenedor del formulario se utilizo el verde indicado en la rubrica con el metodo css tradicional, escrito en el archivo style.css

  3. Clases para margin y padding: Ocupar las clases de Bootstrap que asignan margin y padding acorde a la maqueta del desafío.

    Se ocuparon padding del tipo BOOTSTRAP en las lineas 65 y 236, por el otro lado el margin se ocupo en la linea 249.

  4. Componente Card: Diseñar las Cards de los cupones utilizando como base el componente Card de Bootstrap.

    Se ocupo una componente CARD BOOTSTRAP, copiado de la web de BOOTSTRAP y que fue modificado para agregarle los elementos faltantes como los precios.

  5. Estructura e indentación: Entregar el código del proyecto debidamente indentado y utilizando la siguiente estructura de carpetas: ● /assets/img para las imágenes. ● /assets/css/

    Por ultimo se utilizo la estructura y la identacion, respectando la estructura de carpetas /assets/img para las imágenes. ● /assets/css/
